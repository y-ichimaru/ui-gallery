# vue-ui-gallery

## Project setup
```
yarn install
```

## DEMO

### StoryBookの開発サーバー起動
```
yarn serve
```

### StoryBookのプロダクションビルド

```
yarn build:story
```

ビルドの出力先は/docsです。このディレクトリがGithub Pagesと連動していて、このディレクトリ配下のファイルを書き換えることで、ページを更新します。

[DEMO](https://scene-live.github.io/ui-gallery/)
